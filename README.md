# 🎮 Life As A Skill Tree

A random thought I had:

> What if learning skills felt more like leveling up in a game?

So instead of staring at boring roadmaps and checklists, I built a small RPG-style app that turns career progression into a skill tree.

Pick a path, enter what you already know, and the app shows where you are in the journey.

## 🚀 Running The Project

Clone the repo:

```bash
git clone https://github.com/yourusername/life-as-a-skill-tree.git
```

Move into the folder:

```bash
cd life-as-a-skill-tree
```

Run:

```bash
python main.py
```

---

## 🤔 Why I Made This

Most learning roadmaps look like giant walls of text.

I wanted something that felt a bit more interactive and fun.

Games are good at showing progress.

Learning usually isn't.

This project was basically me trying to combine those two things.
